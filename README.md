# EtsyForge AI

**The AI Operating System for Etsy Digital Product Sellers**

Create, design, and optimize digital products for Etsy with AI. Generate planners, eBooks, workbooks, and printables in minutes.

## 🚀 Features

- **AI Idea Lab** — Generate winning product ideas
- **Design Studio** — Canva-style editor with AI tools
- **Listing Optimizer** — Auto-generate SEO titles, tags, descriptions
- **Asset Library** — Organize all your designs
- **Brand Kit** — Store logos, colors, fonts
- **Etsy Export** — Ready-to-list product packs

## 🛠️ Tech Stack

- Next.js 14 + TypeScript
- Tailwind CSS + shadcn/ui
- Clerk Authentication
- Framer Motion
- Lucide Icons

## 📦 Deployment

This project is ready to deploy on Vercel:

1. Connect your GitHub repository to Vercel
2. Add environment variables (see `.env.local.example`)
3. Deploy!

## 📝 Environment Variables

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```
